Udacity Machine Learning Nanodegree
Capstone Project: Predicting the Daily Direction of the S&P500
By Stephen Fox
October 2016

1. All data used for the project was downloaded from Yahoo! Finance:

http://finance.yahoo.com/quote/%5EGSPC/history?p=%5EGSPC
http://finance.yahoo.com/quote/%5EVIX/history?p=^VIX

The raw data is provided as part of the project submission (SP500_historical.csv)


2. All project code was written Python (2.7.12) in iPython Notebooks:
	Fox-Capstone.ipynb

To run the notebook, type the following from a command line: 
	jupyter notebook Fox-Capstone.ipynb

3. Machine learning libraries used:
	scikit-learn
	TensorFlow
